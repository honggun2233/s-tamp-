# 설문 플랫폼 등록 가이드 — 전자서명 시스템 만족도 조사

---

## 1. Google Forms (구글 폼)

### 접속
- URL: https://forms.google.com
- 구글 계정 로그인 필요

### 설문 생성 순서

#### Step 1 — 새 양식 만들기
1. `forms.google.com` 접속 → **빈 양식** 클릭
2. 상단 제목에 `전자서명 시스템 통합테스트 사용자 만족도 조사` 입력
3. 설명란에 안내문 입력

#### Step 2 — 섹션 구성 (총 6개 섹션)

| 섹션 번호 | 섹션 제목 |
|:---------:|:----------|
| 섹션 1 | 응답자 기본 정보 |
| 섹션 2 | 시스템 접근성 및 사용 편의성 |
| 섹션 3 | 시스템 기능 및 성능 |
| 섹션 4 | 보안 및 신뢰도 |
| 섹션 5 | 기존 업무 대비 개선 효과 |
| 섹션 6 | 종합 만족도 |

> 오른쪽 툴바 → **섹션 추가** 아이콘(=) 클릭으로 섹션 생성

#### Step 3 — 문항 유형별 입력 방법

| 설문 문항 유형 | Google Forms 항목 유형 |
|:-------------|:----------------------|
| 단일 선택 (소속, 직급 등) | **객관식 질문** |
| 5점 척도 | **선형 배율** (1~5, 레이블 입력) |
| 복수 선택 (Q20, Q21) | **체크박스** |
| 자유 기입 (Q22) | **장문형** |
| 조건 분기 (Q11 → Q12) | **섹션으로 이동** 기능 설정 |

#### Step 4 — 조건부 분기 설정 (Q11)
1. Q11 문항 우측 **⋮** → **응답에 따라 섹션으로 이동**
2. "예" → Q12가 있는 섹션으로 이동
3. "아니오" → Q13 섹션으로 이동

#### Step 5 — 설문 배포
1. 상단 오른쪽 **보내기** 버튼 클릭
2. 배포 방법 선택:
   - **링크**: URL 복사 → 카카오톡/이메일 공유
   - **이메일**: 직접 수신자 입력
   - **퍼가기**: HTML 코드 복사 → 사내 인트라넷 삽입
3. **링크 단축** 체크 시 짧은 URL 생성 (goo.gl/forms/...)

#### Step 6 — 응답 확인 및 분석
- 상단 **응답** 탭 클릭
- **요약**: 자동 그래프/통계 확인
- **스프레드시트 아이콘** 클릭 → Google 스프레드시트로 자동 저장
- 스프레드시트에서 성과 평가용 데이터 추출 가능

## 2. Naver Forms (네이버 폼)

### 접속
- URL: https://form.naver.com
- 네이버 계정 로그인 필요 (사내 네이버 계정 또는 개인 계정)

### 설문 생성 순서

#### Step 1 — 새 설문 만들기
1. `form.naver.com` 접속 → **설문 만들기** 클릭
2. 제목: `전자서명 시스템 통합테스트 사용자 만족도 조사`
3. 설명/안내 문구 입력

#### Step 2 — 문항 유형별 입력 방법

| 설문 문항 유형 | Naver Forms 항목 유형 |
|:-------------|:----------------------|
| 단일 선택 | **객관식** |
| 5점 척도 | **척도형** (1~5 자동 설정) |
| 복수 선택 | **복수 응답** |
| 자유 기입 | **주관식(장문)** |

#### Step 3 — 문항 추가
- **문항 추가** 버튼 클릭 → 유형 선택 → 내용 입력
- 섹션 구분: **구분선/소제목** 기능 활용

#### Step 4 — 응답 기간 및 공개 설정
1. **설문 기간** 설정 (시작일/종료일)
2. **응답 제한**: 1인 1회 응답 제한 설정 (로그인 필요 옵션)
3. **공개 범위**: 링크 보유자만 응답 가능으로 설정

#### Step 5 — 설문 배포
1. **배포** 버튼 클릭
2. 생성된 URL 복사 → 메신저/이메일로 공유
3. QR코드 생성 기능 활용 → 출력물에 부착 가능

#### Step 6 — 응답 확인
- **내 설문** → 해당 설문 클릭 → **응답 현황** 탭
- 차트/통계 자동 제공
- **엑셀 다운로드** → 성과 평가 자료로 활용

## 3. 성과 평가 활용 방법

### 점수 산출 기준 (5점 척도 문항 기준)

| 평균 점수 | 등급 | 해석 |
|:---------:|:----:|:-----|
| 4.5 ~ 5.0 | S | 매우 우수 — 고과 최상위 반영 |
| 4.0 ~ 4.4 | A | 우수 — 고과 상위 반영 |
| 3.5 ~ 3.9 | B | 양호 — 고과 평균 반영 |
| 3.0 ~ 3.4 | C | 보통 — 개선 필요 |
| 3.0 미만  | D | 미흡 — 즉각 개선 필요 |

### 고과 반영 권장 항목 (핵심 KPI)

| 항목 | 설문 문항 번호 | 가중치 |
|:-----|:-------------:|:------:|
| 전반적 만족도 | Q19 | 30% |
| 시스템 편의성 | Q4, Q5, Q7 평균 | 25% |
| 업무 효율 개선 | Q16, Q17 평균 | 25% |
| 순추천지수(NPS) | Q18 | 20% |

> **NPS 계산**: Q18에서 5점(적극 권장) 비율 - 1~2점(권장 안 함) 비율

### 엑셀 집계 수식 예시 (Google 스프레드시트 / Excel 공통)

```excel
# 전체 평균 만족도
=AVERAGE(Q19열 범위)

# 고과 점수 계산 (가중 평균)
=AVERAGE(Q19열)*0.3 + AVERAGE(Q4,Q5,Q7열)*0.25 + AVERAGE(Q16,Q17열)*0.25 + AVERAGE(Q18열)*0.2

# NPS 계산
=(COUNTIF(Q18열,"5")/전체응답수) - (COUNTIF(Q18열,"<=2")/전체응답수)
```
